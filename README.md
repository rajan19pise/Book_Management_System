# Book Management System (C Language)

The Book Management System is a console-based application developed in C language to efficiently manage library book records. This project helps automate basic library operations such as adding, viewing, searching, updating, and deleting book details using a simple menu-driven interface.

It uses file handling to store data permanently, ensuring that records are not lost after the program is closed.

---

Features

- Add new book records (ID, title, author, etc.)
- Display all available books
- Search books by ID or title
- Update existing book information
- Delete book records
- Permanent storage using file handling
- Simple and user-friendly menu-driven interface

---

Technologies Used

- C Programming Language
- Structures
- Functions
- File Handling (`fopen`, `fread`, `fwrite`, `fseek`, etc.)
- Standard Input/Output Library
